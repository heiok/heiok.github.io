Q: 如何让Chrome自动启动网页并以全屏模式打开？

A:

  • 启动代码startup.bat

  • 按键代码send_f11.ps1

  • 设置，按下 Win + R → 输入：shell:startup → 回车，把 startup.bat 放进打开的启动文件夹。
